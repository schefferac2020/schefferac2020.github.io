Research

I'm broadly interested in robotics, computer vision, deep learning, and image processing. At the moment, I'm particularly interested in active perception in embodied AI.

I'm currently working as a researcher in the Intelligent Robotics and Autonomy Lab (IRAL) under Prof. Vasileios Tzoumas. Right now, we're developing and testing stable controllers for a drone with additional degrees of freedom for each propellor.

I'm also a summer research assistant in the FROG Lab at UofM under professor Katie Skinner. I'm developing an underwater shipwreck detection pipeline using side-scan sonar imagery and other bathymetric data.

Wreck-tify: Consistent Diffusion Correction For Shipwreck Mapping
Drew Scheffer, Ashwin Saxena, Christian Foreman
Class Project, 2024
presentation / github / pdf

Our team presents various diffusion-based underwater image restoration methods that are used to dehaze and color-correct underwater images such that they are conducive to above-water SLAM and scene reconstruction pipelines.

Da-TRASH: Depth-appended Tabletop Recycling Algorithm for Segmenting Havoc
Drew Scheffer, Ashwin Saxena
Class Project, 2023
project page / github / pdf

Our group extends upon previous work by adapting an existing Unseen Object Instance Segmentation (UOIS) model so that it can perform segmentation on RGB-only images.

Treble in the Sheets: Optical Music Recognition
Drew Scheffer, Ashwin Saxena, Christian Foreman
Class Project, 2022
video / github / pdf

This work presents a simplified optical musical recognition system. Our system takes images of sheet music as input and outputs both an image of the annotated sheet music and a MIDI file for easy parsing
